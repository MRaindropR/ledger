# 📒 SmartLedger V2.8

> 一个完全运行在浏览器本地的个人记账应用，无需服务器、无需安装，单文件即可使用。

---

## ✨ 核心特性

### 零依赖部署
- **单 HTML 文件**，React 18 / SheetJS / 所有依赖全部内联
- 打开即用，无需 Node.js、无需构建、无需网络
- 数据默认存储在浏览器 `localStorage`，刷新不丢失

### 账单管理
- 支持**收入 / 支出 / 转账**三种类型
- 内置丰富分类体系（餐饮、交通、购物、薪资、理财等）
- 商户智能学习：记住同一商户的分类，下次自动填充
- 账单列表支持筛选、搜索、分页

### 账户管理
- 支持**资产账户**（银行卡、证券账户、余额宝等）和**负债账户**（信用卡、花呗等）
- 自定义账户图标、主题色
- 账户拖拽排序、隐藏/显示
- 点击账户查看逐月收支明细和流水

### 数据导入
- 支持从 Excel/CSV 批量导入账单
- 智能字段映射，自动识别日期、金额、说明
- 支持合并同类项（同商户消费与退款对冲）

### 统计报表
- 本月收入、支出、结余一览
- 近 30 天收支趋势折线图
- 月度支出分类饼图 + 明细排行

### 备份与同步
- **导出 Excel 备份**：账户、账单、商户学习记录完整导出
- **导入恢复**：从备份文件一键还原所有数据
- **云同步（Supabase）**：设置自定义密钥，多设备共享同一份数据，每次修改自动同步

### 移动端响应式（V2.8 新增）
- 手机上自动切换为底部 Tab 导航 + 顶部标题栏
- 浮动"+"按钮快速记账
- 首页、账户页、报表页布局全部适配小屏幕

## 💾 数据存储方案

SmartLedger 提供三层数据存储，按优先级依次：

| 层级 | 方式 | 说明 |
|------|------|------|
| 1 | Supabase 云同步 | 需配置密钥，多设备实时同步 |
| 2 | 本地 JSON 文件 | 通过 File System Access API 关联本地文件，跨会话持久化 |
| 3 | localStorage | 默认兜底，同一浏览器内持久 |

### 云同步配置
1. 前往备份页面 → 云同步区域
2. 输入一个自定义密钥（如 `yourname2026`）
3. 点击"保存密钥"，之后每次数据变更自动同步
4. 在其他设备输入同一密钥，点击"从云端恢复"即可获取最新数据

## 🛠 技术栈

| 技术 | 用途 |
|------|------|
| React 18.2 | UI 渲染（CDN 内联，无 JSX 编译） |
| SheetJS | Excel 导入导出 |

| IndexedDB | 跨会话记住文件句柄 |
| localStorage | 默认数据持久化 |
| CSS Media Query | 移动端响应式布局 |

## ⚠️ 注意事项

- 云同步密钥为**明文存储**，请勿使用重要密码作为密钥
- `localStorage` 容量约 5MB，账单数据庞大时建议启用云同步或文件关联
- 导入 Excel 备份会**完全替换**当前数据，操作前请先导出备份
- 应用不收集任何数据，所有数据均由用户自己掌控
